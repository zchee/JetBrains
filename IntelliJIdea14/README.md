IntelliJ Idea 14 Preferences
=====

# Preferences Setup
Backup original file, and make aliases.

```sh
$ sh bootstrap.sh
```

# Application Update
If you intellij you update,

```sh
$sh update.sh
```

make Info.plist, idea.properties and idea.vmoptions aliases only.
